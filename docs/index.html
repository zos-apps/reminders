Reminders

Keep track of tasks with lists and due dates.

GitHub npm

Features

✓ Task lists
✓ Due dates
✓ Priority levels
✓ Recurring reminders
✓ Location alerts

Installation

From App Store

Open the App Store in zOS and search for "Reminders".

Via npm

npm install @zos-apps/reminders

How to Use

  • Create lists
  • Add reminders with +
  • Set due dates

Keyboard Shortcuts

ShortcutAction
Cmd+NNew reminder
Cmd+LNew list

Development

# Install dependencies
npm install

# Start development
npm run dev

# Build for production
npm run build
