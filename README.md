## About

Keep track of tasks with lists and due dates.

## Features

- Task lists
- Due dates
- Priority levels
- Recurring reminders
- Location alerts

## Installation

### From App Store
Open the **App Store** in zOS and search for "Reminders".

### Via npm
```bash
npm install @zos-apps/reminders
```

## Usage

- Create lists
- Add reminders with +
- Set due dates

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | New reminder |
| `Cmd+L` | New list |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
